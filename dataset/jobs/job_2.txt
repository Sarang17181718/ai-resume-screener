Job Title: Cloud Security Engineer
We are seeking a Cloud Security Engineer to design and implement secure cloud solutions across enterprise environments.
Required Skills:
- AWS Security
- Azure Security
- Python
- Linux
- Firewalls
- IAM (Identity and Access Management)
- SIEM Tools
Responsibilities:
- Implement and monitor cloud security controls on AWS and Azure.
- Develop automation scripts in Python for security compliance.
- Configure firewalls and intrusion detection systems.
- Manage IAM roles and policies for secure access.
- Conduct vulnerability assessments and penetration testing.
- Ensure compliance with industry security standards.
Qualifications:
- Bachelor’s degree in Computer Science, Information Security, or related field.
- Experience in cloud security and enterprise risk management.
